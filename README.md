# Student Management System (Java)

A console-based Student Management System implemented in Java.

## Features
- Add new student
- Update student details
- Delete student
- Display all students
- File-based persistence using text file

## Technologies Used
- Java
- File Handling (BufferedReader, BufferedWriter)
- Collections (ArrayList, HashSet)

## Project Structure
- Student.java – Student model
- Services.java – Business logic and file operations
- Main.java – User interaction

## Future Improvements
- Apply OOPS principles (Encapsulation, Abstraction)
- Database integration
- Exception handling improvements

## Run By
-java Main
